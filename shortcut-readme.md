# Shortcut Guide (macOS)

Docmost supports various keyboard shortcuts inspired by a popular note-taking tool.

## Global

- **⌘P** or **⌘K** – Open search spotlight

## Editor

- **⌘B** – Toggle bold
- **⌘I** – Toggle italic
- **⌘U** – Toggle underline
- **⌘E** – Toggle inline code
- **⌘⇧S** – Toggle strikethrough
- **⌘⇧8** – Bullet list
- **⌘⇧7** – Numbered list
- **⌘⇧9** – To‑do list
- **⌥⌘1** … **⌥⌘6** – Heading levels 1‑6
- **⌥⌘0** – Paragraph
- **/** – Open command menu
